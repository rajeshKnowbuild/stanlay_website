Detection Modes

Four ways to find what's buried

Each mode targets a different signal type, giving you complete coverage across all buried utility scenarios.

⚡
Power Mode

Detects buried electricity cables by identifying the mains frequency signal travelling along the cable when current is being drawn. No transmitter required.

50–400 Hz Max 3m depth
📻
Radio Mode

Picks up re-radiated radio signals that are naturally present on metal pipes and cables. Effective for quick surveys without signal generator setup.

10 kHz–30 kHz Max 2m depth
📡
Generator Mode

Uses the SGV4/SGA4 signal generator to apply a 33+131 kHz signal to the target utility, enabling precise route tracing and depth measurement.

33 + 131 kHz Max 5m depth
🔍
All Scan Mode

Simultaneous multi-frequency sweep for comprehensive utility mapping in complex, service-dense areas. Reduces risk of missed utilities.

50 Hz–33 kHz Max 3m depth

FAQ

Common questions

Does DXL4 detect MS Pipe and DI Pipe? +

DXL4 detects MS pipe and all metallic utilities. It cannot detect DI (Ductile Iron) pipe as it is non-metallic in nature.

Does it require periodic calibration? +

No periodic calibration is required. Both the receiver and SGV4 transmitter perform an automatic self-test on first use each day.

Does it have Bluetooth connectivity? +

Bluetooth is available on the DXL4-DBG variant. The standard DXL4 and DXL4-D models transfer data via USB communication port.

Can it be used at night? +

Yes. Both the receiver and SGV4 transmitter feature backlit LCD screens suitable for low-light and nighttime operation.

What is the minimum depth capability? +

Minimum detection depth is 20 cm from ground level. An audible alert signals utilities located at shallow depth.

Why does the DXL4 have data logging? +

Data logging records date, time, modes used, and usage intervals. This gives project managers compliance evidence that avoidance surveys were conducted before excavation.

Can rechargeable batteries be used? +

Rechargeable batteries can be used but Stanlay recommends high-quality non-rechargeable batteries such as Duracell for optimal 40-hour operating time.

What does 131 kHz frequency do? +

131 kHz is optimised for detecting thin buried utilities — including small-bore cables and sheathed fibre optic ducts — that lower frequencies may miss.
